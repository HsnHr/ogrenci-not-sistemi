# Student Grade Management System

A simple Java console application using Object-Oriented Programming (OOP).

##  Features
- Add students with grades
- Calculate average grade
- Show pass/fail status
- List students

##  Technologies
- Java
- OOP
- ArrayList

##  How to Run
- Open project in Java IDE
- Run `OgrenciNotSistemi.java`

##  Purpose
Practicing Java, OOP and basic programming concepts.
